
🐭 Mimi – AI Desktop Companion

Your AI friend that lives on your desktop.

Mimi is an AI-powered desktop companion designed to make everyday computer use more engaging, organized, and enjoyable. Instead of being just another chatbot, Mimi acts like a friendly virtual companion that chats naturally, provides emotional support, helps manage daily tasks, tracks productivity, stores personal diary entries, and offers fun mini-games—all while living on your desktop as an animated character.

## Run Locally

**Prerequisites:**  Node.js

🚀 Getting Started

Clone the repository:

git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git

Install dependencies:

npm install

Create a .env.local file:

GEMINI_API_KEY=YOUR_GEMINI_API_KEY

Start the development server:

npm run dev
